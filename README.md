# sektor-rehberim
İş hayatına yeni başlayanlar için anonim şirket inceleme ve mülakat deneyimi paylaşım platformu. Kullanıcılar departman bazlı (Yazılım, İK vb.) gerçek deneyimlerini paylaşır, mülakat sorularını listeler ve her sayfa için dinamik QR kod oluşturabilir. React &amp; Firebase ile geliştirilen, etik odaklı bir rehber uygulaması.
